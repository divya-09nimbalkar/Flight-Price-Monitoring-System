# Flight Price Monitoring System - Documentation

## Purpose
This project monitors flight prices, cleans the data, analyzes trends, and visualizes results.

## Modules
- **scraper.py**: fetches flight data (mock or API).
- **processor.py**: cleans and transforms data.
- **analytics.py**: computes stats.
- **visualization.py**: generates plots.
- **main.py**: orchestrates the pipeline.

## Next Steps
- Integrate real flight API (e.g., Skyscanner, Amadeus).
- Add ML models for price prediction.
- Expand notebooks for exploratory analysis.
